# 8:1 Multiplexer Hardware Implementation

##  Description
A hardware implementation of an 8-to-1 Multiplexer using digital logic ICs, switches, LEDs, and a custom soldered PCB. The system selects one of eight input signals and transfers it to a single output based on the select line combination.

##  Objectives
- Understand multiplexer operation
- Implement digital logic hardware
- Verify truth table practically
- Learn PCB soldering and circuit testing

##  Components Used
- 8:1 Multiplexer IC (74LS151 or equivalent)
- Toggle Switches
- LEDs
- Current Limiting Resistors
- PCB Board
- Connecting Wires
- 5V Power Supply

##  Working Principle
1. Eight switches provide input signals (D0–D7)
2. Three select lines (A, B, C) choose one input
3. Multiplexer processes the selected input
4. Output LED indicates the selected input state
5. Truth table is verified through hardware testing

##  Features
